Es un programa sencillo en donde el usuario, coloca las temperaturas de la semana de lunes a domingo y el progrmaa calcula el promedio semanal de las temperaturas 
Con la finalidad de explicar la diferencias de la programación tradicional con POO, se crea el programa

Diferencias 
Con la programación tradicional, el código se ve más largo y desordenado, en cambio con la programación POO el código se estructuró en clases, lo que permite tener un mejor control del código y orden 
Otra de las ventajas es la simplicidad y mantenimiento.
